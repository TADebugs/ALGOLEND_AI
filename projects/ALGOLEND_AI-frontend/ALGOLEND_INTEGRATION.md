## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Connect Wallet**: Click "Connect Wallet" to connect your Algorand wallet

4. **Explore Features**: Navigate through different tabs to explore the interface
